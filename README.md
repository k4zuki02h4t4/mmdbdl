# mmdbdl
Shellscript to download mmd files from Maxmind.

# How to Use

Beforehand, obtain your LICENSE KEY from the **[MaxMind account page](https://www.maxmind.com/en/accounts/current/license-key)**.

Usage is very simple.
After cloning the repository, just give execute permission to the shell script.

```bash
$ git clone https://github.com/k4zuki02h4t4/mmdbdl.git
$ cd ./mmdbdl
$ chmod +x mmdbdl.sh

```

Here is how to actually use it:

```bash
$ sudo ./mmdbdl.sh YOUR_ACCOUNT_ID YOUR_LICENSE_KEY

```
